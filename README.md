# Jovo CLI

Command Line Interface for the [Jovo Framework](https://github.com/jovotech/jovo-framework-nodejs)


* [Getting Started](#getting-started)
* [Commands](#commands)


## Getting Started

```
$ npm install -g jovo-cli
```

Verify by looking up the version:
```
$ jovo -V
```


## Commands


### Create a new project

```
$ jovo new <directory>
```
